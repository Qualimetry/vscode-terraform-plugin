Enforce the official Terraform JSON syntax with a root object (keys: resource, variable, output, etc.). Array-at-root syntax is not the documented standard.

Noncompliant Code Example

[{"resource": {"aws_instance": {"a": {}}}}]

Compliant Solution

{"resource": {"aws_instance": {"a": {}}}}
