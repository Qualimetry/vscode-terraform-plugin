Disallow deprecated 0.11-style interpolation. Interpolation-only expressions like `"${var.x}"` are redundant in Terraform 0.12+; use `var.x` instead.

Noncompliant Code Example

resource "aws_instance" "a" {
  ami = "${var.ami}"
}

Compliant Solution

resource "aws_instance" "a" {
  ami = var.ami
}
