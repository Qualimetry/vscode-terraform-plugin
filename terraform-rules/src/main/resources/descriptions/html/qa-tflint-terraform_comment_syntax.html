Enforce usage of `#` for comments. The Terraform language supports `#`, `//`, and `/* */`; `#` is considered idiomatic for both single and multi-line comments.

Noncompliant Code Example

// single-line comment
/* multi-line comment */

Compliant Solution

# Use # for single and multi-line comments
