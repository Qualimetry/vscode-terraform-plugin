Protocol must be valid for the security group rule.

Noncompliant Code Example

resource "aws_security_group_rule" "a" { type = "ingress"; protocol = "bad"; security_group_id = "sg-123" }

Compliant Solution

resource "aws_security_group_rule" "a" { type = "ingress"; from_port = 0; to_port = 0; protocol = "icmp"; security_group_id = "sg-123" }
