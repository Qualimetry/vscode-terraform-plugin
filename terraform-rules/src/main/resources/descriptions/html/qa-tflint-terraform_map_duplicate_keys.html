Disallow duplicate keys in a map. Duplicate keys are overwritten by Terraform and usually indicate a mistake.

Noncompliant Code Example

locals {
  m = { a = 1, a = 2 }
}

Compliant Solution

locals {
  m = { a = 1, b = 2 }
}
