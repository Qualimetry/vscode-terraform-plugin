# Ensure MWAA environment is not publicly accessible

`qa-checkov-CKV2_AWS_66` &middot; Security &middot; Vulnerability &middot; severity CRITICAL

## Summary

Ensure MWAA environment is not publicly accessible.

## Noncompliant code example

```hcl
resource "aws_s3_bucket_acl" "a" { bucket = aws_s3_bucket.a.id; acl = "public-read" }
```

## Compliant solution

```hcl
resource "aws_s3_bucket_acl" "a" { bucket = aws_s3_bucket.a.id; acl = "private" }
```
