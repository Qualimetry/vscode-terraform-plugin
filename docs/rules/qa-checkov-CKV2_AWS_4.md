# Ensure API Gateway stage have logging level defined as appropriate

`qa-checkov-CKV2_AWS_4` &middot; Security &middot; Vulnerability &middot; severity MAJOR

## Summary

Ensure API Gateway stage have logging level defined as appropriate.

## Noncompliant code example

```hcl
resource "aws_s3_bucket" "a" { bucket = "my-bucket" }
```

## Compliant solution

```hcl
resource "aws_s3_bucket" "a" { bucket = "my-bucket"; logging { target_bucket = aws_s3_bucket.logs.id; target_prefix = "log/" } }
```
