# Ensure Kinesis Stream is securely encrypted

`qa-checkov-CKV_AWS_43` &middot; Security &middot; Vulnerability &middot; severity MAJOR

## Summary

Ensure Kinesis Stream is securely encrypted.

## Noncompliant code example

```hcl
resource "aws_ebs_volume" "a" { size = 10; availability_zone = "us-east-1a" }
```

## Compliant solution

```hcl
resource "aws_ebs_volume" "a" { size = 10; availability_zone = "us-east-1a"; encrypted = true }
```
